🚀 NeuroFleetX
AI-powered fleet management system with real-time tracking, booking, and role-based dashboards.

🛠 Tech Stack
React (Frontend)
Spring Boot (Backend)
MySQL (Database)
Leaflet + OSRM (Maps & Routing)
✨ Features
Vehicle Booking System
Real-time Fleet Tracking
Role-Based Dashboards
KPI Analytics
📂 Project Structure
/frontend → React App
/backend → Spring Boot API
⚙️ Setup Instructions
Backend
cd backend
mvn spring-boot:run
