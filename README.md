# 🚀 NeuroFleetX

NeuroFleetX is an AI-powered fleet management and urban mobility system designed to optimize vehicle booking, tracking, and operational efficiency using real-time data and role-based dashboards.

---

## 📌 Features

- 🚗 Smart Vehicle Booking System  
- 📍 Real-Time Fleet Tracking (Map Integration)  
- 🔐 Secure Authentication using JWT  
- 👥 Role-Based Access (Customer, Driver, Admin, Manager)  
- 📊 Live Dashboards with KPIs & Analytics  
- 🔄 Booking Lifecycle (CONFIRMED → COMPLETED)  
- ⚡ Dynamic Vehicle Availability (No Double Booking)  

---

## 🛠️ Tech Stack

### 💻 Frontend
- React.js  
- React Leaflet (Maps)  
- Recharts (Charts & Analytics)  

### ⚙️ Backend
- Java Spring Boot  
- REST APIs  
- Spring Security + JWT  

### 🗄️ Database
- MySQL  

### 🌐 APIs
- OpenStreetMap (Maps)  
- OSRM API (Routing & Distance Calculation)
